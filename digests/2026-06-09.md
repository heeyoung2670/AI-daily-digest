## 🧪 연구 및 논문 동향

### "소프트웨어 공학의 종말": AI 에이전트가 소프트웨어 패러다임을 재구성한다 (arXiv:2606.05608)
LLM이 1차 추론 엔진으로 동작하며 코드를 도구로 동적으로 생성·폐기하는 AI 에이전트의 등장이 소프트웨어 공학의 근본 전제를 해체하고 있다고 주장하는 논문이다. 사양 작성·설계·구현·테스트 같은 기존 단계가 의미를 잃어가는 이유를 분석하며, 개발자의 역할이 "코드 작성자"에서 "에이전트 지시·감독자"로 전환되고 있음을 이론화한다.  
출처: https://arxiv.org/abs/2606.05608

### 멀티모달 LLM-as-a-Judge의 지각적 판단 편향 완화 (arXiv:2606.02578)
시각 증거와 텍스트가 충돌할 때 멀티모달 LLM 평가자(judge)가 지각적으로 올바른 답 대신 그럴듯한 서사에 점수를 주는 편향이 있음을 발견했다. 지각적 섭동(perceptual perturbation)과 보상 모델링 기법을 결합해 이 편향을 줄이는 방법을 제안한다. 자동화 평가 파이프라인의 신뢰도를 높이는 데 직접 적용 가능하다.  
출처: https://arxiv.org/abs/2606.02578

### BAGEN: LLM 에이전트는 예산을 인식하는가?
LLM 에이전트가 API 호출 비용이라는 예산 제약을 실제로 인식하고 행동을 조정하는지 체계적으로 평가하는 벤치마크다. 대부분의 현재 에이전트가 비용 의식 없이 과도한 툴 호출을 한다는 결과를 제시하며, 경제적으로 효율적인 에이전틱 시스템 설계에 대한 연구 공백을 부각한다.  
출처: https://arxiv.org/list/cs.AI/current
