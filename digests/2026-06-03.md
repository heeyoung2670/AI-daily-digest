## 🧪 연구 및 논문 동향

### Unified Context Evolution for LLM Agents (arXiv:2606.02304)
LLM 에이전트는 에피소드마다 동일한 고정 컨텍스트에서 출발해 유용한 전략을 습득해도 태스크 종료 시 소실되는 한계가 있었다. UCE(Unified Context Evolution)는 에이전트 경험을 메모리·전략·워크플로·스킬 4가지 유형의 Evolvable Context Unit(ECU)으로 외부 라이브러리에 저장해 점진적으로 진화시키는 그래디언트-프리 프레임워크다. 별도 파인튜닝 없이 에이전트 성능을 지속 향상시킬 수 있어 실용적 배포 환경에 적합하다.

- 출처: https://arxiv.org/abs/2606.02304

### 멀티모달 LLM 공간 추론의 공간 어휘 편향 진단 (arXiv:2606.01914)
9개 오픈소스 멀티모달 LLM을 대상으로 기계적 해석 가능성 도구를 적용해 공간 추론 실패를 분석했다. 답안 선택지에 공간 관계어(위, 아래, 왼쪽 등)를 추가하는 것만으로 모델 결정이 바뀌는 "공간 어휘 편향" 현상을 발견했다. 비전-언어 모델의 공간 추론 신뢰성을 평가할 때 간과하기 쉬운 취약점을 규명한 연구다.

- 출처: https://arxiv.org/abs/2606.01914

### Small Language Models are the Future of Agentic AI (arXiv:2506.02153)
NVIDIA 연구진이 작성한 논문으로, 에이전틱 시스템에서 대부분의 호출은 전문화된 소규모 태스크를 반복 수행하므로 범용 대형 모델보다 소형 언어 모델(SLM)이 더 비용 효율적이라고 주장한다. 범용 대화 능력이 필요한 경우 여러 SLM을 조합한 이종 에이전트 시스템이 최선임을 논증하며, 에이전트 인프라 설계 방향에 실질적인 시사점을 제공한다.

- 출처: https://arxiv.org/abs/2506.02153
