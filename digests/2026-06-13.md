### 2. Anthropic, "Claude가 자사 코드의 80% 작성" 공개 선언 및 글로벌 AI 일시정지 제안
Anthropic이 공개한 "When AI Builds Itself" 보고서에서 Claude가 이미 Anthropic 프로덕션 코드의 80% 이상을 작성하고 있다고 밝혔다. 이를 '재귀적 자기 개선(recursive self-improvement)'의 조기 신호로 규정하며, 다른 프론티어 랩이 동의할 경우 개발 속도를 검증 가능한 방식으로 늦추거나 일시 정지하자고 제안했다.
- 출처: https://themenonlab.blog/blog/anthropic-recursive-self-improvement-ai-builds-itself-june-2026/

### 3. MiniMax M3 오픈소스 가중치 공개 — SWE-Bench Pro 59.0% 오픈웨이트 SOTA
6월 1일 API로 먼저 선보인 MiniMax M3의 모델 가중치가 오픈소스로 공개됐다. MSA(MiniMax Sparse Attention) 아키텍처로 1M 토큰 컨텍스트를 지원하며 SWE-Bench Pro에서 59.0%를 기록해 오픈웨이트 최고치를 달성했다. 입력 $0.60 / 출력 $2.40 per million tokens 가격도 경쟁력이 높다.
- 출처: https://www.marktechpost.com/2026/06/01/minimax-releases-minimax-m3-with-msa-architecture-supporting-1m-token-context-native-multimodality-and-agentic-coding/

### 4. Snap, 직원 1,000명 해고 — "AI가 코드 65% 생성, 연 $5억 비용 절감"

- 출처: https://www.crescendo.ai/news/latest-ai-news-and-updates

## 🧪 연구 및 논문 동향

### 광학적 추론(Optical Reasoning): 이미지를 텍스트 대신 추론 매체로 활용 (arXiv:2606.09585)
LLM이 추론 과정에서 텍스트 대신 이미지·도형·수식을 시각적 추론 단계로 직접 생성·활용하는 'Optical Reasoning' 프레임워크를 제안한다. 수학·과학·다중 모달 추론 벤치마크에서 텍스트 기반 CoT(Chain-of-Thought)와 동등하거나 이를 초과하는 성능을 보여, 멀티모달 추론 연구의 방향을 새롭게 제시한다.
- 출처: https://arxiv.org/abs/2606.09585

### LLM 에이전트 통신 프로토콜의 보안 위험 서베이 — MCP·A2A 중심 (arXiv:2506.19676)
MCP(Anthropic)와 A2A(Google) 같은 에이전트 통신 프로토콜이 확산되면서 발생하는 새로운 보안 위협(프롬프트 인젝션, 권한 탈취, 중간자 공격)을 체계적으로 분류하고 방어 전략을 제시한 서베이. 에이전트 기반 서비스를 구축하는 개발자에게 실질적인 위협 모델 수립에 직접 활용 가능하다.
- 출처: https://arxiv.org/abs/2506.19676

### 소형 언어 모델이 에이전틱 AI의 미래다 (arXiv:2506.02153)
대형 모델을 오케스트레이터로, 소형 특화 모델을 실행 에이전트로 사용하는 계층형 멀티에이전트 아키텍처가 비용·지연·프라이버시 측면에서 단일 대형 모델보다 우수함을 실험으로 입증한다. 엣지 디바이스·온프레미스 환경에서 에이전트를 배포하려는 개발자에게 참고할 만한 설계 원칙을 제공한다.
- 출처: https://arxiv.org/abs/2506.02153
