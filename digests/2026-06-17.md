# AI Daily Digest — 2026-06-17

## 🔥 주요 헤드라인

### 1. Anthropic, 미국 정부 긴급 명령으로 Fable 5·Mythos 5 전면 접근 차단
미국 상무부가 6월 12일 긴급 수출 통제 명령을 발동해 Anthropic에 Claude Fable 5·Mythos 5의 해외 접근을 즉시 차단하도록 지시했다. Anthropic은 "실시간으로 내·외국인을 구분할 방법이 없다"며 전 세계 이용자 대상 서비스를 전면 중단했다. 회사는 해당 취약점이 공개 모델로도 재현 가능한 수준이라 반박하면서도 명령에 따랐다. 나머지 Claude 모델은 정상 운영 중이다.
- 출처: https://www.cnbc.com/2026/06/12/anthropic-disables-access-to-fable-5-and-mythos-5-to-comply-with-government-directive.html

### 2. OpenAI, ChatGPT에 'Lockdown Mode' 전체 배포 — 프롬프트 인젝션 데이터 탈취 차단
6월 4~8일에 걸쳐 개인·사업자 계정에 Lockdown Mode가 배포됐다. 활성화 시 아웃바운드 네트워크 요청, 웹 브라우징, Deep Research, 에이전트 모드가 비활성화된다. 민감한 데이터를 다루는 팀이 프롬프트 인젝션 공격으로 인한 데이터 유출을 막기 위한 선택적 보안 레이어다. 메모리·파일 업로드 기능은 그대로 유지된다.
- 출처: https://techcrunch.com/2026/06/06/openai-unveils-lockdown-mode-to-protect-sensitive-data-from-prompt-injection-attacks/

### 3. Google, DiffusionGemma 26B 오픈소스 공개 — 기존 대비 텍스트 생성 4배 빠름
6월 10일 Google DeepMind가 확산(Diffusion) 방식의 26B MoE 오픈소스 모델을 출시했다. 순차적 토큰 생성 대신 텍스트 블록을 병렬로 생성해 H100에서 1,000+ 토큰/초, RTX 5090에서 700+ 토큰/초를 기록한다. Apache 2.0 라이선스로 Hugging Face·Kaggle·Vertex AI에서 무료로 이용 가능하며, NVIDIA 플랫폼에 최적화됐다. 단, 표준 벤치마크에서 Gemma 4보다 성능이 낮은 트레이드오프가 있다.
- 출처: https://mlq.ai/news/google-deepmind-releases-diffusiongemma-a-26b-open-source-model-that-generates-text-4x-faster-via-diffusion/

### 4. Meta, 인도 Reliance와 168MW AI 데이터센터 계약 — 연간 capex $1,250억~$1,450억
6월 9~10일 Meta가 Reliance Industries와 인도 구자라트주 자므나가르에 168MW 규모 AI 데이터센터 임대 계약을 체결했다. Meta의 인도 첫 AI 인프라 투자다. 재생에너지 파트너로 CleanMax·Fourth Partner Energy와 1GW 규모 계약도 병행했다. 2026년 전체 자본 지출 전망치는 $1,250억~$1,450억으로 상향됐다.
- 출처: https://techcrunch.com/2026/06/10/meta-signs-first-ai-data-center-deal-in-india-with-reliance/

### 5. Gemini CLI, 내일(6/18) 종료 — Antigravity CLI(agy)로 즉시 전환 필요
Google이 Gemini CLI를 6월 18일 소비자·무료 티어 기준으로 종료한다. 후계 도구 Antigravity CLI(agy)는 Node.js 대신 Go로 재작성됐으며 비동기 백그라운드 에이전트 워크플로를 지원한다. API 키 대신 OAuth 브라우저 인증을 사용한다. CI/CD 스크립트에서 `gemini` 명령을 `agy`로 교체해야 하며, 기업용 Standard·Enterprise 라이선스는 영향을 받지 않는다.
- 출처: https://developers.googleblog.com/an-important-update-transitioning-gemini-cli-to-antigravity-cli/

LLM이 수학 문제를 구조화된 스키마로 변환한 뒤 SymPy를 이용해 결정론적으로 검증하는 파이프라인이다. 자기일관성 루프나 비평 모델 없이 단일 패스로 답을 내고 검증까지 마친다. "신뢰 우선(Trust-First)" 설계로 LLM의 수학 환각 문제를 구조적으로 차단한다.
- 출처: https://arxiv.org/abs/2606.00671

에이전틱 RL이 LLM의 패러다임을 텍스트 생성에서 목표 설정·장기 계획·동적 전략 적응으로 전환한다고 분석한 포괄적 서베이다. 멀티에이전트 협업, 외부 도구 활용, 자기개선 루프 등 핵심 설계 패턴을 정리하고 현재 한계와 미래 방향성을 제시한다.
- 출처: https://arxiv.org/abs/2604.27859
