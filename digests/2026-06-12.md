## 🧪 연구 및 논문 동향

### CyberGym: 실제 취약점으로 AI 에이전트 사이버보안 능력 평가 (arXiv:2506.02548)
LLM 에이전트가 실제 CVE 취약점을 얼마나 자율적으로 탐지·익스플로잇할 수 있는지 측정하는 대규모 벤치마크. 현재 최상위 모델도 복잡한 멀티스텝 공격 체인에서는 성공률이 낮지만, 단순 취약점 스캐닝에서는 숙련 보안 연구자 수준에 근접함을 확인했다.
- 출처: https://arxiv.org/abs/2506.02548

### LLM 에이전트 통신과 MCP: 소프트웨어 설계 패턴 중심 서베이 (arXiv:2506.05364)
Model Context Protocol(MCP)이 에이전트 간 통신 표준으로 급부상한 배경을 클래식 소프트웨어 설계 패턴 관점에서 분석한 서베이. Pub/Sub, Mediator, Command 패턴 등이 MCP 설계에 어떻게 투영됐는지 정리하며, 멀티에이전트 시스템의 신뢰성·확장성 확보 방법론을 제시한다.
- 출처: https://arxiv.org/abs/2506.05364

### SafeTuneBed: LLM 파인튜닝 후 안전 정렬 깨지는 현상 벤치마크 (arXiv:2506.00676)
파인튜닝이 LLM의 안전 정렬을 어느 정도 손상시키는지 체계적으로 측정하는 툴킷. 감성 분석·QA·멀티스텝 추론 등 다양한 파인튜닝 태스크와 방어 기법(면역화, 훈련 중 세이프가드, 사후 복구)을 통합 평가한다. 커스텀 파인튜닝 서비스를 구축하는 개발자에게 직접적인 참고 자료가 된다.
- 출처: https://arxiv.org/abs/2506.00676
