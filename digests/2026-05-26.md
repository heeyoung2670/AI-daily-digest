# 2026-05-26 AI Daily Digest

## 🔥 주요 헤드라인

### 1. Meta, Muse Spark 공개 — 오픈소스 10년 역사에 마침표
Meta가 Alexandr Wang(ScaleAI 창업자, Meta Superintelligence Labs 수장)이 주도한 첫 독점 모델 Muse Spark를 출시했다. Llama 시리즈로 대표되던 10년간의 오픈소스 노선을 공식 폐기한 것으로, 훨씬 적은 컴퓨트로 Llama 4 대비 동급 성능을 달성하며 멀티모달 인식·추론·에이전틱 작업에서 프론티어 경쟁력을 갖췄다. "향후 오픈소스화를 희망한다"는 단서를 달았지만 즉시 공개 계획은 없다.
- 출처: https://techcrunch.com/2026/04/08/meta-debuts-the-muse-spark-model-in-a-ground-up-overhaul-of-its-ai/

### 2. Anthropic + Google + Broadcom — 3.5GW 컴퓨트 파트너십, Google $40B 투자
Anthropic이 Google·Broadcom과 TPU 기반 컴퓨트 3.5GW를 2027년부터 단계적으로 확보하는 파트너십을 체결했다. Google은 $40B 규모 투자(즉시 $10B, 성과 조건부 $30B)도 병행했다. ARR이 1분기 동안 $9B에서 $30B+로 급증하며 수요가 공급을 초과한 결과다.
- 출처: https://www.anthropic.com/news/google-broadcom-partnership-compute

### 3. GPT-5.5 Instant — ChatGPT 기본 모델 교체, 할루시네이션 52% 감소
OpenAI가 5월 5일 GPT-5.5 Instant를 전체 ChatGPT 사용자에게 배포하며 GPT-5.3 Instant를 교체했다. 고위험 프롬프트(의료·법률·금융)에서 할루시네이션 52.5% 감소, 사용자 신고 오류 37.3% 감소가 핵심 지표다. 과거 대화·파일·Gmail을 참조하는 강화된 개인화 기능도 기본 탑재됐다.
- 출처: https://openai.com/index/gpt-5-5-instant/

### 4. Gemini 3.5 Flash GA — 4배 빠른 프론티어 모델, API 정식 출시
Google I/O 2026(5/19)에서 공개된 Gemini 3.5 Flash가 API GA 상태로 전환됐다. 1M 토큰 컨텍스트, $1.50/$9.00 per 1M 토큰, Terminal-Bench 2.1 76.2%로 전작 Gemini 3.1 Pro를 모든 코딩·에이전틱 벤치마크에서 앞선다. Google AI Studio, Vertex AI, Gemini 앱에서 즉시 사용 가능하다.
- 출처: https://blog.google/innovation-and-ai/models-and-research/gemini-models/gemini-3-5/

### 5. Claude Code, Colossus 1 계약으로 rate limit 2배 증가
Anthropic이 xAI의 Colossus 1(GPU 22만 개 이상) 렌탈 계약을 맺고 5월 6일 Claude Code 사용 한도를 전면 2배로 올렸다. 5시간 한도 2배 증가, 피크타임 페널티 완전 폐지. API Tier 1 기준 분당 토큰 한도는 30K → 500K으로 약 16배 상향됐다.
- 출처: https://hundredtabs.com/blog/claude-code-rate-limits-doubled-2026

## 🧪 연구 및 논문 동향

### On-Policy Data Evolution for Visual-Native Multimodal Deep Search Agents (arXiv:2605.10832)
오픈 월드 문제를 검색·툴 사용·시각적 추론을 연쇄적으로 엮어 해결하는 멀티모달 딥서치 에이전트 연구. ODE(On-Policy Data Evolution) 프레임워크를 적용한 결과 Qwen3-VL-8B 기반 에이전트가 8개 멀티모달 딥서치 벤치마크 평균에서 24.9% → 39.0%로 크게 향상됐다. 에이전트 학습 데이터를 정책에 맞게 동적으로 진화시키는 것이 핵심이다.
- 출처: https://arxiv.org/abs/2605.10832

### MemEye: A Visual-Centric Evaluation Framework for Multimodal Agent Memory (arXiv:2605.15128)
장기 에이전트 메모리가 점점 멀티모달해지고 있지만 기존 평가 기준은 텍스트 중심이라는 문제를 지적한 논문. '결정적 시각 증거의 세분성(씬 레벨 → 픽셀 레벨)'과 '증거 활용 방식(단일 → 진화적 합성)' 두 축으로 메모리 능력을 평가한다. RAG 기반 에이전트의 비주얼 메모리 취약점을 구체적으로 드러낸다.
- 출처: https://arxiv.org/abs/2605.15128

### Predictive Maps of Multi-Agent Reasoning: Successor-Representation Spectrum for LLM Communication Topologies (arXiv:2605.11453)
체인·스타·메시 등 멀티 에이전트 LLM 시스템의 커뮤니케이션 토폴로지를 후계자 표현(Successor Representation) 스펙트럼으로 통합 분석한 논문. 어떤 토폴로지가 어떤 태스크 유형에 최적인지 예측하는 지도(predictive map)를 제시한다. 멀티 에이전트 설계에서 토폴로지 선택의 이론적 근거를 마련했다.
- 출처: https://arxiv.org/abs/2605.11453

## 🏢 기업 소식

### Meta — Muse Spark, 오픈소스 포기와 Superintelligence Labs 전략
Muse Spark는 Llama 4 미드사이즈 대비 "한 자릿수 배" 적은 컴퓨트로 동급 성능을 달성한다. Facebook·Instagram·WhatsApp·Ray-Ban 안경에 우선 탑재되며, 향후 오픈소스화 가능성은 열어두었지만 구체적 일정이 없다. 이 결정은 Meta가 소비자 AI 생태계를 직접 통제하겠다는 의도로 읽힌다.
- 출처: https://venturebeat.com/technology/goodbye-llama-meta-launches-new-proprietary-ai-model-muse-spark-first-since

### ElevenLabs — $500M ARR 돌파, 엔터프라이즈 음성 에이전트 수요 폭증
2025년 말 $350M ARR에서 2026년 4개월 만에 $500M ARR을 달성했다. 성장 동력은 고객 지원·영업·채용·마케팅 전반에 음성 에이전트를 배포하는 엔터프라이즈 수요다. 텍스트 AI가 성숙기에 접어드는 사이 음성 에이전트 인프라가 새로운 ARR 성장축으로 부상하고 있다.
- 출처: https://elevenlabs.io/blog/500m-arr-and-new-investors

### Google — Anthropic에 $40B 투자 확정, AI 생태계 주도권 강화
Google이 Anthropic에 즉시 $10B을 투자하고, 성과 조건부로 추가 $30B을 투자하는 딜을 확정했다. Anthropic 기업가치는 $350B으로 평가됐다. Google Cloud를 통한 Anthropic 모델 서빙이 확대되면서 Google은 자체 Gemini와 외부 클로드 모두를 클라우드로 공급하는 양면 전략을 구사하고 있다.
- 출처: https://techcrunch.com/2026/04/24/google-to-invest-up-to-40b-in-anthropic-in-cash-and-compute/

## 🛠️ 개발자 도구 및 오픈소스

### OpenCode — 오픈소스 코딩 에이전트 1위, GitHub 160K stars 돌파
2025년 6월 출시 후 11개월 만에 GitHub 160K+ stars를 달성해 Claude Code(122K+)를 앞질렀다. 75개 이상 AI 제공사·로컬 모델을 선택적으로 스왑할 수 있으며, LSP(언어 서버 프로토콜) 통합으로 컴파일러 오류를 실시간 피드백해 자동 수정한다. 7.5M 월간 활성 개발자, 900명+ 기여자로 커뮤니티 생태계가 빠르게 성장 중이다.
- 출처: https://agentconn.com/blog/opencode-open-source-ai-coding-agent-review-2026/

### Gemini 3.5 Flash API — 개발자 마이그레이션 가이드 공개
gemini-3.5-flash 모델 ID로 Google AI Studio와 Vertex AI에서 즉시 사용 가능. 이전 gemini-2.0-flash 대비 추론 능력이 대폭 향상됐으며 컨텍스트 윈도우는 1M 토큰. 캐시 입력은 $0.15/1M, 멀티모달 입력 지원. Gemini API 공식 변경 이력과 마이그레이션 가이드가 함께 공개됐다.
- 출처: https://ai.google.dev/gemini-api/docs/changelog

### Claude Code rate limits 상향 — Colossus 1 GPU 22만 개 계약
Anthropic이 xAI Colossus 1 슈퍼컴퓨터를 임대하며 인퍼런스 용량을 대폭 늘렸다. Pro/Max/Enterprise 플랜 모두 5시간 한도 2배 증가, 피크타임 패널티 제거. API에서는 Tier에 따라 최대 16배까지 한도가 늘었으며, 스마트 라우팅 활성화 시 동일 비용으로 5배 더 많은 요청을 처리할 수 있다.
- 출처: https://www.aiforanything.io/blog/claude-code-rate-limits-doubled-may-2026-update
