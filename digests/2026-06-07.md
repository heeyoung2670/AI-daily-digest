### 02. OpenAI 모델, 에르되시 이산기하학 추측 반증 — 80년 묵은 수학 난제 해결
OpenAI의 범용 추론 모델이 1946년 헝가리 수학자 에르되시(Erdős)가 제시한 '단위 거리 추측'을 반증하는 무한 계열 반례를 스스로 찾아냈다. 골로드-샤파레비치 이론과 무한 유체탑 이론을 활용한 대수 정수론 기법이 핵심이며, 필즈 메달리스트 팀 가워스는 "AI 수학의 이정표"로 평가했다. 프린스턴 수학자 윌 소윈이 검증 후 개선된 n^(1+0.014) 경계를 확정했다.
- 출처: https://openai.com/index/model-disproves-discrete-geometry-conjecture/

## 🧪 연구 및 논문 동향

### DeployBench: LLM 에이전트의 연구 결과물 배포 능력 평가 (arXiv:2606.05238)
AI·ML, 컴퓨터 시스템, 과학 컴퓨팅 등 3개 도메인에 걸쳐 51개 연구 코드 배포 태스크를 모은 멀티도메인 벤치마크. OpenHands 기반으로 4개 최신 LLM을 평가하면 통과율이 7.8%~51.0%에 그쳐, 현행 에이전트가 실제 연구 환경 배포에서 여전히 크게 부족함을 실증한다.
- 출처: https://arxiv.org/abs/2606.05238

### Agent Planning Benchmark (APB): 에이전트 계획 능력 종합 진단 (arXiv:2606.04874)
22개 도메인, 4,209개 멀티모달 사례로 LLM 에이전트의 계획 능력을 측정하는 진단 프레임워크. 전체 계획, 피드백 조건부 단계별 계획, 불필요 도구·고장 도구·해결 불가 태스크에서의 강건성 등 5가지 설정을 포괄한다. 기존 에이전트 벤치마크들이 계획 단계를 별도로 분리하지 않았다는 한계를 체계적으로 보완한다.
- 출처: https://arxiv.org/abs/2606.04874

### Latent Reasoning with Normalizing Flows (NF-CoT) (arXiv:2606.06447)
연속적인 '잠재 사고 흐름'을 정규화 흐름(normalizing flows)으로 모델링하는 추론 프레임워크 NF-CoT 제안. 토큰화된 Chain-of-Thought의 이산 병목을 제거해 추론 경로의 표현력을 높이며, 기존 Latent CoT 방식 대비 수학·코딩 벤치마크에서 일관된 성능 향상을 보고한다.
- 출처: https://arxiv.org/abs/2606.06447

## 🛠️ 개발자 소식

### ADK Arena: LLM-as-Developer로 에이전트 개발 키트 비교 평가 (arXiv:2606.05548)
사람 개발자 대신 LLM 코딩 에이전트가 문서를 학습하고 에이전트 코드를 작성·수정하는 방식으로 여러 에이전트 개발 키트(ADK)를 평가하는 프레임워크. 실제 개발자 경험을 자동화해 ADK 간 생산성을 객관적으로 비교할 수 있어, 프레임워크 선택 시 참고 자료로 활용 가능하다.
- 출처: https://arxiv.org/abs/2606.05548

### Search-Time Contamination: 딥 리서치 에이전트의 벤치마크 오염 연구 (arXiv:2606.05241)
웹 검색을 사용하는 딥 리서치 에이전트가 공개 벤치마크 정답을 검색으로 우회해 성능이 인위적으로 부풀려지는 '검색 시점 오염(Search-Time Contamination)' 현상을 체계적으로 분석한 논문. 공개 벤치마크 기반 에이전트 평가의 신뢰성에 근본적 의문을 제기하며, 오프라인·프라이빗 평가 방식으로의 전환 필요성을 강조한다.
- 출처: https://arxiv.org/abs/2606.05241

### MemoryDocDataSet: 대화 기억 + 장문 문서 추론 결합 벤치마크 (arXiv:2606.04442)
50개 마이크로 월드와 1,000개 QA 쌍으로 구성된 합성 벤치마크로, LLM이 대화 맥락 기억과 긴 문서 추론을 동시에 처리하는 능력을 측정한다. 현행 모델들이 두 작업을 분리하면 잘 하지만 결합하면 성능이 뚜렷하게 떨어짐을 보여, 멀티-태스크 메모리 연구의 새 기준점을 제시한다.
- 출처: https://arxiv.org/abs/2606.04442
