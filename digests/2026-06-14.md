### 4. Google, Gemini CLI → Antigravity CLI 전환 발표 — 6월 18일 기존 서비스 종료
Google이 Gemini CLI를 Antigravity CLI(agy)로 교체하며 에이전트 플랫폼을 전면 재편했다. Antigravity 2.0은 데스크톱 앱·CLI·SDK·Managed Agents API를 하나로 묶은 플랫폼 전략으로, 6월 18일부터 기존 Gemini CLI·Gemini Code Assist IDE 확장은 서비스를 중단한다. Cursor를 넘보는 에이전틱 IDE 플랫폼으로의 전환이다.
- 출처: https://developers.googleblog.com/an-important-update-transitioning-gemini-cli-to-antigravity-cli/

## 🧪 연구 및 논문 동향

### Gemini 3의 UI 에이전트 정확도 7배 도약 — ScreenSpot-Pro 72.7%
멀티모달 AI의 화면 이해 능력이 극적으로 개선되었다. Gemini 3는 ScreenSpot-Pro 벤치마크에서 72.7%를 기록해, Gemini 2.5 Pro의 11.4% 대비 약 6.4배 향상됐다. 실제 소프트웨어 UI를 클릭·탐색하는 에이전트 자동화의 실용화가 빠르게 앞당겨지고 있다.
- 출처: https://medium.com/@adityaj5400/beyond-text-the-rise-of-large-multimodal-models-a-2026-deep-dive-0843292fa048

### "더 생각할수록 덜 보는" 역설 — 멀티모달 추론 모델의 환각 연구
멀티모달 추론 모델이 더 긴 체인-오브-소트(CoT)를 생성할수록 시각 정보를 덜 참조하고 환각이 증가한다는 연구 결과가 arXiv에 발표됐다. 텍스트 추론 능력을 높이는 강화학습이 시각 인식 능력과 충돌할 수 있다는 점에서, 멀티모달 에이전트 설계 시 주의가 필요하다.
- 출처: https://arxiv.org/abs/2505.21523

### ICML 2026 — LLM의 자기 역량 평가(Capability Self-Assessment) 연구
ICML 2026에 발표된 논문에서는 LLM이 자신의 능력 범위를 얼마나 정확히 인식하는지를 측정하는 새로운 벤치마크가 제안됐다. 모델이 "할 수 없다"는 것을 알면서도 시도하거나, 역으로 할 수 있는 일을 거부하는 오류 패턴을 정량화했다. 에이전틱 시스템에서 모델 자기 인식의 중요성이 부각된다.
- 출처: https://arxiv.org/list/cs.AI/current

### Google Antigravity CLI — Gemini CLI 대체, 코드·웹·파일 자율 실행 에이전트
터미널에서 `agy` 명령으로 실행되는 Antigravity CLI는 코드 실행·웹 탐색·파일 관리를 격리 샌드박스에서 자율 수행한다. Gemini 3.5 Flash 기반으로 프리 티어를 포함해 개인 개발자도 무료로 사용 가능하다. 기존 Gemini CLI 사용자는 6월 18일 전에 전환이 필요하다.

Claude Code 최신 버전에서 하나의 에이전트가 하위 에이전트를 중첩 생성하는 서브에이전트 기능이 추가됐다. 스마트 모델·리전 자동 선택, MCP 플러그인 검색, Chrome 연동 강화가 포함된다. Agent SDK 분리 과금 전환과 함께 Claude Code의 기능 범위가 본격적인 프로덕션 에이전트 플랫폼 수준으로 확장되는 중이다.
