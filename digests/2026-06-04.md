### 03. Claude Opus 4.8 성능 상세 — SWE-bench Verified 88.6%, Fast Mode 3배 저렴
5월 28일 출시된 Claude Opus 4.8가 SWE-bench Verified 88.6%(전작 대비 1% 향상), SWE-bench Pro 69.2%(GPT-5.5보다 10점 이상 앞선)를 기록했다. Fast Mode 요금이 이전 세대 대비 3배 저렴해졌고, Claude Code의 새 "dynamic workflows" 기능으로 대규모 엔지니어링 문제를 병렬 서브에이전트로 처리할 수 있다. 기본 가격은 Opus 4.7과 동일하게 입력 100만 토큰당 $5 / 출력 $25.

## 🧪 연구 및 논문 동향

### Imaginative Perception Tokens Enhance Spatial Reasoning in Multimodal LMs (arXiv:2606.03988)
멀티모달 언어 모델의 공간 추론 약점을 해결하는 논문으로 CVPR 2026 MUSI 워크숍에 발표됐다. 모델이 관찰된 시각 장면을 기반으로 "대안적 공간 배치를 상상한 중간 인식 표현"(IPT)을 생성하도록 학습시킨다. 이를 통해 별도 외부 도구 없이 VLM 자체의 공간 추론 정확도를 크게 개선할 수 있음을 보였다.

- 출처: https://arxiv.org/abs/2606.03988

### Governed Collaborative Memory as Artificial Selection in LLM-Based Multi-Agent Systems (arXiv:2605.04264)
LLM 다중 에이전트 시스템에서 기억 지속성을 "인공 선택"에 비유해 설계한 논문이다. 비관리 지속성·헌법적 선택·자동 메트릭 기반 선택·인간 승인 선택 4단계로 기억 거버넌스를 분류하고, 에이전트 로컬 메모리·공유 기관 메모리·아카이브·프로젝트 연속성 메모리로 계층화된 아키텍처를 제안한다. 장기 실행 에이전트 시스템 설계에 실질적인 지침을 제공한다.

- 출처: https://arxiv.org/abs/2605.04264

### Ares: Adaptive Reasoning Effort Selection for Efficient LLM Agents (arXiv:2603.07915)
LLM 에이전트가 확장된 Chain-of-Thought 추론을 사용할 때 더 깊은 환경 분석이 가능하지만 추론 비용이 크게 증가하는 딜레마를 다룬다. 입력 복잡도에 따라 동적으로 추론 노력을 조절하는 Ares 프레임워크를 제안해 성능 손실 없이 토큰 소비를 대폭 줄인다. 비용 민감 에이전트 배포 환경에서 즉시 적용 가능한 실용적 연구다.

- 출처: https://arxiv.org/abs/2603.07915
