### 4. OpenAI, ChatGPT 기본 모델을 GPT-5.5 Instant로 조용히 교체

## 🧪 연구 및 논문 동향

### LLM 에이전트 메모리 메커니즘 진화 서베이 (arXiv:2605.06716)
**"From Storage to Experience: A Survey on the Evolution of LLM Agent Memory Mechanisms"**

LLM 기반 에이전트의 메모리 시스템이 어떻게 발전해왔는지를 세 단계로 체계화한 서베이 논문이다. ① Storage(궤적 보존) → ② Reflection(궤적 정제) → ③ Experience(궤적 추상화)로 이어지는 진화 프레임워크를 제안한다. 장기 일관성 유지, 동적 환경 대응, 지속 학습이라는 세 가지 핵심 과제가 이 진화를 이끈다고 분석한다. 실용적 관점에서 에이전트 메모리 설계를 이해하기 좋은 참고 자료다.
- 출처: [arXiv:2605.06716](https://arxiv.org/abs/2605.06716)

### LLM 에이전트 벤치마크 12종 투명성 감사 (arXiv:2605.21404)
**"What Twelve LLM Agent Benchmark Papers Disclose About Themselves: A Pilot Audit"**

주요 LLM 에이전트 벤치마크 12개 논문을 대상으로 평가 실행 방법에 대해 실제로 어떤 정보를 공개하는지 감사했다. 재현 가능성, 평가 기준 명확성, 데이터 오염 여부 검토 방법 등을 오픈 스코어링 스키마로 정량화했다. 공개된 벤치마크라도 실제 평가 조건의 투명성이 크게 부족하다는 점을 드러낸다.
- 출처: [arXiv:2605.21404](https://arxiv.org/abs/2605.21404)

### 에이전틱 AI 워크로드 특성 분석 (arXiv:2605.26297)
**"Agentic AI Workload Characteristics"**

ReAct 스타일 에이전트를 LLM 서빙 측면과 도구 실행 측면 두 관점에서 분석했다. Gemma, Qwen 모델의 추론·비추론 설정과 5개 아젠틱 벤치마크에 걸쳐 종단 간 트레이싱 인프라를 구축해 워크로드 특성을 측정했다. 에이전트 시스템의 병목이 어디에 있는지 파악하는 데 유용한 기준 데이터를 제공한다.
- 출처: [arXiv:2605.26297](https://arxiv.org/abs/2605.26297)
