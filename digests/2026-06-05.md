## 🧪 연구 및 논문 동향

### Deliberate Evolution: 아젠틱 추론으로 심볼릭 회귀 효율 40% 개선 (arXiv:2606.04360)
LLM 기반 심볼릭 회귀(수식 발견)의 비효율성을 다루는 논문이다. 기존 방식이 수식 생성과 탐색 제어를 하나의 프롬프트에 혼합해 샘플 비효율이 심했다는 점에 착안, DE(Deliberate Evolution) 프레임워크는 이 두 역할을 분리한다. 방향 탐색 연산자, 구조 분석 도구, 경험 메모리를 통해 LLM이 제안 방향을 명확히 인식하도록 설계됐다. LLM-SRBench에서 기존 대비 동일 성능을 표준 샘플 예산의 40%만으로 달성했다.
- 출처: https://arxiv.org/abs/2606.04360

### 멀티모달 LLM 공간 추론의 어휘 편향 메커니즘 분석 (arXiv:2606.01914)
멀티모달 LLM의 공간 추론 실패가 시각적 처리가 아닌 언어 측(language side)에서 기인한다는 것을 기계론적 해석 도구로 밝혔다. '위', '아래', '왼쪽' 같은 공간 관계 단어를 선택지에 추가하는 것만으로 모델의 답변이 유도되는 "공간 어휘 편향(spatial lexical bias)"을 최초로 정의하고 발생 위치를 특정 뉴런·채널 수준에서 추적했다.
- 출처: https://arxiv.org/abs/2606.01914

### 멀티모달 LLM-as-a-Judge의 지각 편향 완화 (arXiv:2606.02578)
멀티모달 LLM을 평가자(judge)로 사용할 때, 시각적 증거와 텍스트 단서가 충돌하면 모델이 시각적으로 올바른 답보다 "그럴듯한 서술"을 선호하는 편향을 분석했다. 지각 교란(perceptual perturbation)과 보상 모델링을 결합해 이 편향을 완화하는 방법을 제안한다. AI 생성 콘텐츠 평가 파이프라인 설계에 직접 활용할 수 있는 결과물이다.
- 출처: https://arxiv.org/abs/2606.02578
