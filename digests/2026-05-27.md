## 🧪 연구 및 논문 동향

### EvoSafety: 외재화된 공격-방어 공진화로 LLM 안전성 확보 (arXiv:2605.13411)
기존 LLM 안전 정렬은 모델 가중치에 내재돼 있어 새로운 공격에 취약하다는 문제를 다룬다. EvoSafety는 외부에 지속·검사·재사용 가능한 구조를 만들어 LLM 안전성을 모델 불가지론적으로 유지한다. 방어 성공률 99.61%를 달성하면서도 일반 쿼리의 추론 성능은 유지했다.
- 출처: https://arxiv.org/abs/2605.13411

### Cattle Trade: LLM 블러핑·입찰·협상 멀티에이전트 벤치마크 (arXiv:2605.14537)
경매, 숨겨진 가격 거래, 협상, 블러핑, 자원 배분을 결합한 멀티에이전트 벤치마크를 제안한다. 전략적 기만과 합리적 협상이 동시에 요구되는 환경에서 현재 LLM의 한계를 체계적으로 평가할 수 있다. 에이전틱 AI의 게임이론적 행동 연구의 새로운 기준선을 제시한다.
- 출처: https://arxiv.org/abs/2605.14537

### ARGUS: 컨텍스트 인식 프롬프트 인젝션으로부터 LLM 에이전트 방어 (arXiv:2605.03378)
웹 브라우징·도구 사용 에이전트가 악의적인 외부 콘텐츠에 노출될 때 발생하는 프롬프트 인젝션 공격을 방어하는 프레임워크다. 에이전트의 실행 컨텍스트를 추적해 공격 신호를 탐지하며, 다양한 에이전트 아키텍처에 적용 가능한 모델 불가지론적 설계를 채택했다.
- 출처: https://arxiv.org/abs/2605.03378

### ARGUS 오픈소스 공개 — 프롬프트 인젝션 방어 라이브러리
arXiv 논문 2605.03378과 함께 ARGUS 라이브러리 코드가 공개됐다. LLM 에이전트에 미들웨어 방식으로 삽입해 외부 콘텐츠로부터의 프롬프트 인젝션을 감지·차단한다. 실무 에이전트 앱을 보호하려는 개발자에게 바로 적용 가능한 솔루션이다.
- 출처: https://arxiv.org/abs/2605.03378

75개 이상 AI 제공사·로컬 모델 스왑 지원, LSP 통합으로 컴파일러 오류 실시간 자동 수정을 지원하는 OpenCode가 Claude Code(122K+ stars)를 제치고 오픈소스 코딩 에이전트 1위에 올랐다. 월 활성 개발자 7.5M, 기여자 900명+ 규모로 성장했다.
