## 🧪 연구 및 논문 동향

### Predictive Maps of Multi-Agent Reasoning (arXiv 2605.11453)
멀티에이전트 LLM 시스템에서 통신 토폴로지(체인·스타·메시 등)를 선택하기 전에 성능 특성을 예측하는 진단 도구를 제안한다. Successor Representation 이론을 기반으로 어떤 토폴로지가 드리프트·합의·강건성에서 어떻게 작동하는지 사전 추론 없이도 판단할 수 있다. 멀티에이전트 파이프라인을 설계하는 개발자에게 실용적인 참고 자료.
- 출처: https://arxiv.org/abs/2605.11453

### AgentRx: LLM 에이전트 기반 멀티모달 임상 예측 벤치마크 (arXiv 2605.10286)
전자의무기록(EHR)·영상·리포트·임상 메모를 함께 처리하는 LLM 에이전트를 실제 대규모 데이터로 체계 평가한 연구다. 단일 에이전트 vs 멀티에이전트, 단일 모달 vs 멀티모달 조합을 비교하며, 임상 AI 에이전트의 현재 한계와 가능성을 가늠하는 데 유용하다.
- 출처: https://arxiv.org/abs/2605.10286

### Less Is More: 수학 추론에서 프롬프트 복잡도의 한계 (arXiv 2604.18897)
40개 이상의 프롬프트 변형을 GPT·Llama·Gemma 3가지 모델에 걸쳐 실험한 결과, 단순하고 짧은 프롬프트가 복잡한 구조적 프롬프트보다 수학 추론 성능이 높은 경우가 많았다. "더 많이 설명할수록 더 잘 이해한다"는 통념을 뒤집는 실증 연구.
- 출처: https://arxiv.org/abs/2604.18897

## 🏢 기업 소식

### Google — Antigravity 2.0: 에이전트 우선 개발 플랫폼 전면 개편
데스크톱 앱·CLI·SDK·Managed Agents(Gemini API)·엔터프라이즈 플랫폼 5가지 구성으로 출시됐다. 단일 API 호출로 도구 사용·코드 실행이 가능한 격리된 Linux 환경에서 에이전트를 구동할 수 있으며, 기존 Gemini CLI도 Antigravity CLI로 전환된다. Google Cloud와 직접 연동되는 엔터프라이즈 배포 경로도 포함됐다.
- 출처: https://www.marktechpost.com/2026/05/19/google-launches-antigravity-2-0-at-i-o-2026-a-standalone-agent-first-platform-with-cli-sdk-managed-execution-and-enterprise-support/

Karpathy는 프리트레이닝 책임자 Nick Joseph 산하에서 "Claude로 프리트레이닝 연구를 가속화"하는 전담 팀을 신설한다. Eureka Labs(AI 교육 스타트업) 경험을 바탕으로 AI 교육 분야 작업도 이어갈 계획이라고 밝혔다.
- 출처: https://www.axios.com/2026/05/19/anthropic-openai-karpathy-andrej-claude

## 🛠️ 개발자 도구 및 오픈소스

### Gemini 3.5 Flash — 지금 바로 쓸 수 있는 GA 모델
API ID: `gemini-3.5-flash` (GA, preview 접미사 없음). 입력 $1.50/백만 토큰, 출력 $9.00/백만 토큰. 컨텍스트 윈도우 1M 입력/65K 출력. vLLM이나 자체 스택 없이 Google AI Studio·Gemini API로 즉시 사용 가능하다. Gemini 3.1 Pro 대비 코딩·에이전트 벤치마크에서 앞선 성능을 보인다.
- 출처: https://simonwillison.net/2026/May/19/gemini-35-flash/

### Antigravity CLI — Gemini CLI의 후속
터미널 워크플로우 개발자를 위한 경량 에이전트 생성 도구. 기존 Gemini CLI 사용자는 Antigravity CLI로 마이그레이션 안내를 받게 된다. Antigravity SDK와 연동해 커스텀 에이전트 동작을 정의하고 원하는 인프라에 호스팅할 수 있다.
- 출처: https://developers.googleblog.com/an-important-update-transitioning-gemini-cli-to-antigravity-cli/

### Google I/O 2026 개발자 하이라이트 — AI Studio·Android Studio 업데이트
AI Studio에 Managed Agents 통합, Android Studio에 에이전트 기반 코드 지원 강화가 포함됐다. Google의 개발자 도구 전체가 Gemini 3.5 Flash와 Antigravity 에이전트 하네스를 기반으로 재편되는 중이다.
- 출처: https://blog.google/innovation-and-ai/technology/developers-tools/google-io-2026-developer-highlights/
