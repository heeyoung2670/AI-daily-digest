## 🧪 연구 및 논문 동향

### Organize then Retrieve: 에이전트 계층적 메모리 탐색 (arXiv:2606.11680)
LLM 에이전트가 장기 작업(long-horizon task) 수행 시 상태를 유지하지 못해 입력 컨텍스트가 무한정 길어지는 문제를 해결하는 계층적 메모리 탐색 기법을 제안한다. 기존의 손실 압축(lossy compression)이나 유사도 기반 검색이 시간적 구조와 인과 관계를 놓치는 한계를 극복하고, 작업 메모리를 계층적으로 조직화해 에이전트 추론 품질과 비용을 동시에 개선한다. 멀티스텝 에이전틱 워크플로우를 구현하는 개발자에게 실용적인 메모리 아키텍처 참고 자료다.

출처: https://arxiv.org/abs/2606.11680

### LLM 에이전트는 세계 모델을 추론할 수 있는가? (arXiv:2606.16576)
LLM 기반 에이전트가 내부 세계 모델을 추론하는 능력을 가지고 있는지를 실험적으로 검증한 연구다. 추론 특화 모델(reasoning model)이 일반 모델보다 훨씬 강하지만, 쿼리 계획 수립과 증거 통합 단계에서 반복적인 실패를 보인다는 점을 밝혔다. 현재 에이전트들이 여전히 얼마나 강건하지 못한지를 정량적으로 보여주는 유용한 벤치마크다.

출처: https://arxiv.org/abs/2606.16576

### LLM 에이전트 통신 프로토콜 기술 분류 체계 (arXiv:2606.19135)
MCP, A2A, OpenAPI 등 빠르게 확산되는 LLM 에이전트 간 통신 프로토콜을 기술적으로 체계화한 서베이 논문이 공개됐다. 에이전트 시스템을 구축하는 개발자라면 어떤 프로토콜이 어떤 상황에 맞는지 판단하는 기준점으로 활용할 수 있다. 프로토콜 표준화 경쟁이 프레임워크 수준에서 본격화되고 있음을 보여주는 시의적절한 자료다.
