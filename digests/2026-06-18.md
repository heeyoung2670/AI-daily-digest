### 3. Gemini CLI, 오늘(6/18) 종료 — Antigravity CLI(agy)로 즉시 전환 필요
Google이 Gemini CLI를 오늘부로 소비자·무료 티어에서 공식 종료한다. 후계자 Antigravity CLI(agy)는 Go로 재작성돼 비동기 백그라운드 에이전트 워크플로와 OAuth 인증을 지원한다. CI/CD 파이프라인이나 쉘 스크립트에서 `gemini` 명령을 사용 중이라면 오늘 중으로 `agy`로 교체해야 한다.
- 출처: https://developers.googleblog.com/an-important-update-transitioning-gemini-cli-to-antigravity-cli/

## 🧪 연구 및 논문 동향

### Arbor: 가설 트리 반복 정제로 자율 연구 수행 (arXiv:2606.11926)
중국인민대 연구팀이 발표한 Arbor는 인간 감독 없이 가설 생성 → 코드 작성 → 실험 실행 → 결과 반영 사이클을 반복하는 에이전틱 연구 프레임워크다. Hypothesis Tree Refinement(HTR)를 통해 이전 실패를 기억해 이후 가설의 출발점을 높인다. MLE-Bench Lite에서 GPT-5.5 기반 86.36% Any Medal을 달성, Codex·Claude Code 대비 평균 2.5배 이상의 상대적 성과 향상을 보였다.
- 출처: https://arxiv.org/abs/2606.11926

### 풀-듀플렉스 음성 LLM의 다면적 상호작용 정렬 (arXiv:2606.11167)
실시간 양방향(full-duplex) 음성 대화 모델이 사람처럼 자연스럽게 끊고 반응하는 "다면적 상호작용성(Multi-Faceted Interactivity)"을 정의하고 평가 프레임워크를 제안한 연구다. 텍스트 중심 정렬 방법론을 음성 LLM에 적용할 때의 한계를 분석하고 음성 전용 정렬 기법 방향성을 제시했다.
- 출처: https://arxiv.org/abs/2606.11167
